Back-end Development Internship

We are currently seeking a motivated, hands-on back-end web development intern. You’ll be responsible for building, testing, debugging, and deploying content management systems, data management systems, and data aggregation systems. You’ll be working along side other front-end developers and designers both in-house and remote to create succinct solutions for the interwebs.
You have already:
  • created awesome things with PHP, MySQL, maybe even with open-source code
  • know your way around a code editor TextEdit, Coda, TextMate, or Espresso
  • written and debugged Javascript with or without using a framework like jQuery
  • worked alongside others to take a project from planning to a working website
  • experimented with a third party API (that’s not a drug)
  • found GitHub and StackOverflow and realized how awesome they are
  • discovered and consume most of the articles on Hacker News
We won’t hold it against you, but it would be awesome if you:
  • know basic UNIX commands and are capable of using the command line
  • have created a theme, modules, or an entire site with an open-source platform like WordPress, Drupal, or maybe even Expression Engine
  • used version control and better yet, used GIT (if not we’ll teach you)
  • know how to write emails that use proper capitalization & punctuation
  • hated everything that applications like Dreamweaver stand for

Ready to start? Awesome, send an email to jobs@developedsimple.com with a little bit about yourself, a few web projects you've worked on, and your Twitter handle. No phone calls please.
